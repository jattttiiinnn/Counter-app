# Counter App

This is a simple counter application built with HTML, Tailwind CSS, and JavaScript.

## Features

- Increment and decrement the counter value
- Reset the counter to zero

## Technologies Used

- HTML
- Tailwind CSS
- JavaScript

## Setup

1. Clone the repository or download the source code.
2. Open the `index.html` file in your browser to view the app.

## Usage

- Click the "Increment" button to increase the counter value.
- Click the "Decrement" button to decrease the counter value.
- Click the "Reset" button to reset the counter to zero.
