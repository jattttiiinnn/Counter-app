# Counter App

This is a simple counter application built with HTML, Tailwind CSS, and JavaScript.

## Features

- Increment and decrement the counter value
- Reset the counter to zero

## Technologies Used

- HTML
- Tailwind CSS
- JavaScript

## Live Demo

The app is deployed and can be accessed at: (https://countmeup-three.vercel.app/)

## Setup

1. Clone the repository or download the source code.
2. Open the `index.html` file in your browser to view the app.

## Usage

- Click the "+" button to increase the counter value.
- Click the "-" button to decrease the counter value.

## Deployment

The application is deployed using Vercel's hosting platform.

## File Structure

- `index.html` - Main HTML file
- `script.js` - JavaScript logic for counter functionality
